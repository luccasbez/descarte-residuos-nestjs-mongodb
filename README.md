# Descarte Resíduos - API (NestJS + MongoDB)

**Estrutura pronta para uso com MongoDB (Mongoose).**

Passos para rodar:
1. Copie `.env.example` para `.env` e substitua `<db_password>` pela sua senha.
2. `npm install`
3. `npm run start:dev`
4. Endpoints principais:
   - POST /pontos
   - GET  /pontos
   - POST /descartes
   - GET  /descartes?usuario=&pontoId=&tipo=&data=YYYY-MM-DD
   - GET  /relatorio
